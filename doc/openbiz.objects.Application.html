Class: Application

openbiz.objects. Application

The Application object is containner of an openbiz based server side application.

Each application module should based on this class to extend it's own feature. The application itself can provides basic methods to get its controllers and models

For easy to access, this class also alias as openbiz.Application.

Normally the first thing after you instance an application class is call its loadToRoute methods to load itself to express router. Otherwise even you have inited an openbiz application, you still cannot access it.

Example
// below sample show you how to create a new application model which extends this module.
module.exports = function(openbiz)
{
    if(typeof openbiz != 'object') return null;
    var application = new openbiz.Application({
        _context : openbiz.context,
        _name : require('path').basename(__dirname),
        _path : __dirname,
        openbiz: openbiz
    });
    return application;
}
